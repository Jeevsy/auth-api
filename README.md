## REST api server starter (using JWT express and mongoose)

### Technologies used:

`JWT`, `express`, `mongoose`

### How to start:

0. 安装mongodb ， nodemon
1. start mongodb: `mongod --dbpath YourDBPath`;
2. in `config.js`, change `DBPATH` to `YourDBPath`, set your `SECRET`
3. run the server: `npm start`
